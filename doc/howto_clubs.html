Clubs Howto

 

How clubs work / what features are present


The clubs feature work like this:

1) The pilot submits his flight as usual to leonardo, there is no difference at this stage.

2) The flight is displayed in the "main" leoanrdo, and counted in the general scoring / xc league.

3) If the pilot is a member of a club (or more clubs) his flight will also appear in the scoring/league
of his club. There is the option for clubs not to count each amd every flight of their members
but give the club admin the possibility to add/delete flights manually. See "addManual" option below.

This is the basic functionallity that is present at the moment.

Things/features to be done

  • noSpecificMembers: If set to 1 the club admin will not have to add pilots manually. Instead pilots will automatically belong to a club based on some criteria (nationalty or DHV category of their glider etc...)
  • defaultDisplayTheme A club may have a totally different apearance by using a diefferent theme.
  • countryCodes Flights will be included only if done in specific countries
  • pilotNationality Pilots will become automatically members of the club only if of specific natianality/nationalities
  • gliderCat Flights will be included only if done with specific DHV category gliders
  • Start - End period : Only flights during a period will be counted

 

Configuration

Edit file site/config_clubs.php There are the club definitions. Here is a typical entry:

 
1=>array(	
 "desc"=>"Broken Thermal Club",
 "id"=>1,
 "lang"=>"mexican",
 "areaID"=>0,
 "addManual"=>1,
 "adminID"=>76,
 ) ,

 

1=>array(
1 is the club ID, no clubs can have the same ID.

"desc"=>"Broken Thermal Club",
The description that will apear on the menu. Try to be brief, and not exceed 40-50 letters.

"id"=>1,
The Id of the club as before

"lang"=>"mexican",
The default language that will be displayed.
Different clubs can have different default languages,usefule in a multinational leonardo server

"areaID"=>0,
Not used at the moment, leave it 0. It will be used when there is possibility to define Areas as
a group of takeoffs and restrain the club fligths to takeoffs of this area

"addManual"=>1,
The most imporant setting: If is set to 0, all the flights that are submited to leonardo by club
members are counted in the club league.
If set to 1, a member must submit his flight to leonardo, but wont be automatically added to the club.
The admin must manually add the flight via a special interface.

"adminID"=>76,
The user id of the administrator of this club. The club admin can add/remove pilots to the club
and if "addManual" (see above ) is set to 1 he can also add/delete flights for the club.

 

You can add as many entries as this , each with a different clubID.
